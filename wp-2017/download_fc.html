In order to use FLAGCAL you need to install PGPLOT and CFITSIO on your computer. Installing CFITSIO is trivial but PGPLOT may be pain (on non-ubuntu systesm). The main issues which I know are as following:

  1. It does not work with the GFORTARN compiler which is most common at present and is distributed with GCC. Getting a version of g77 which is compatible to your latest gcc compiler is next to impossible. Installing an older version of gcc compatible with g77 which you have is non-trivial. I have found a solution which is as follows:

    Go inside PGPLOT source directory and look for "sys_linux" directory: 

    cd sys_linux
    do the following 
    cp  g77_gcc.conf  gfortran_gcc.conf
cp  g77_gcc_aout.conf gfortran_gcc_aout.conf
    now replace g77 from files gfortran_gcc.conf and gfortran_gcc_aout.conf by gfortran (use vi etc.) After doing all that come to the directory where you have your driver file and use the option: 
    linux gfortran_gcc
    after that follow the pgplot docs.
